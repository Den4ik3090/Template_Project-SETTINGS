# Учебные проекты  по веб-разработке! Практика начинающего FRONTEND разработчика!
Не судите строго - над всеми проектами работал начинающий разработчик со стажем 5 месяцев на момент создания репозитория )

Добро пожаловать в репозиторий с учебными заданиями по JavaScript! Здесь собраны различные задачи и мини-проекты, выполненные в процессе обучения.
---
## Структура проекта
- Все задания находятся в папке:  
  `project-setup-template/src/tasks`  
  Каждое задание оформлено в отдельной папке или файле с понятным номером и названием.
- Главная точка входа для сборки и запуска:  
  `project-setup-template/src/index.js`  
  В этом файле вы можете переключать активное задание, раскомментируя импорт нужного модуля.
---
## Используемые технологии и библиотеки
- **JavaScript (ES6+)** — основа логики и разработки.
- **Webpack 5** — сборщик проекта, упрощающий работу с модулями, стилями и оптимизацию.
- **Sass (SCSS)** — препроцессор для управления стилями.
- **Bootstrap 5** — CSS-фреймворк для удобной и быстрой верстки компонентов.
- **Tailwind CSS** — утилитарный CSS для гибкой стилизации.
- **Font Awesome** — иконки для удобства интерфейсов.
- **Babel** — транспиляция современного JS для совместимости с браузерами.

#  Как запустить и просмотреть задания

1. Клонируйте репозиторий:
git clone <URL вашего репозитория>
cd project-setup-template

text
2. Установите зависимости:
npm install

text
3. Запустите локальный сервер для разработки с горячей перезагрузкой:
npm run dev

text
4. В файле `src/index.js` переключите активное задание, раскомментировав импорт нужного модуля из папки `src/tasks`.
5. Откройте в браузере страницу http://localhost:3000 (или порт, указанный в консоли).

## Сборка финальной версии

Для создания оптимизированного продакшн-билда используйте:

npm run build
Готовая версия будет в папке `dist`, которую можно загрузить на любой статический хостинг.
